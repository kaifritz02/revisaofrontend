Módulo 3 - Comandos

As operações do Git são executadas por meio de comandos, geralmente relacionado a operação que você deseja que o Git execute.

COMANDOS PRINCIPAIS:

  • git --version [comando usado para verificar a versão do seu git.]

  • git config --local user.name "seu nick do GitHub"[comando usado para cadastrar o seu usuário do seu GitHub ao Git]

  • git config --local user.email "seu email cadastrado do GitHub"[comando usado para cadastrar o seu e-mail do seu GitHub ao Git]

  • mk dir [comando usado para criar uma pasta local no seu armazenamento interno]

  • cd [usado para navegar entre as pastas do seu computador]

  • git init [comando usando para iniciar um repositório local na sua máquina]

  • git history [verifica e lista todos os comandos que foram usados]

  • git status [verifica o estado atual do seu repositório, bem como os arquivos que foram alterados ou implementados]

  • git commit (ou git commit -m "sua mensagem", podendo inserir uma mensagem junto ao commit) [armazena as alterações ou implementações no seu repositório]
